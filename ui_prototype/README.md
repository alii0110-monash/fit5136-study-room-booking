# UI Prototype (LucidChart)

## Screens

### Member A (Screens 1, 3, 5 + Exception Screens)
- **Screen 1**: Welcome & Authentication (with admin branch)
- **Screen 3**: Account & History Page
- **Screen 5**: Checkout Page (with Deal Package notification)
- **Exception**: Student ID already registered
- **Exception**: Insufficient balance

### Member B (Screens 4, 7, 8 + Exception Screens)
- **Screen 4**: Filter & Booking Page (with Double-check)
- **Screen 7**: Add Room (with equipment loop input)
- **Screen 8**: Delete Room (with WARNING confirmation)
- **Exception**: No matching rooms
- **Exception**: Double-check failure
- **Exception**: Delete rejection (future booking exists)

### Member C (Screens 2 & 6 + Integration)
- **Screen 2**: Student Main Menu
- **Screen 6**: Administrator Main Menu
- **Stitching**: Connect all screens with arrows
- **Text Audit**: Verify against User Stories
- **Usability Annotations**: Principle 1/2/3 labels
- **Legend**: Colors, fonts, arrow meanings
- **START HERE**: Label on Screen 1
- **Closed Loop**: Logout arrows back to Screen 1

## Export Instructions
1. Create LucidChart document
2. Design each screen as separate frames
3. Export as images or shareable link
4. Add navigation arrows between screens
5. Include legend and usability annotations
